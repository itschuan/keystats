# Lite Manual Test Tasks

- Launch Lite for the first time and verify the permission guide is shown
- Verify that missing permission does not show misleading zero data
- Click through to System Settings and manually grant Input Monitoring
- Return to the app after authorization and verify statistics start
- Type in multiple apps and verify Top Apps updates
- Verify the menu bar count updates while typing
- Verify the last 7 days trend displays correctly
- Switch statistics mode and verify detail mode requires a second confirmation
- Clear local data and verify there is a second confirmation and the UI refreshes
- Revoke Input Monitoring permission and verify the UI enters the permission failure state
- Restart the app and verify state recovery is correct
- Test dark / light mode
- Test long App name display
- Run the App Sandbox + CGEventTap feasibility spike
- Prepare and manually review App Review Notes and privacy policy copy
